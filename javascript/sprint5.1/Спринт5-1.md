# Четвертый спринт. Введение



Вы изучили всё важное о вёрстке. Это отличный багаж навыков, но главная часть учебной программы — JavaScript — ещё впереди. До этого момента вы занимались разметкой, теперь займётесь программированием.

</div>

<div class="paragraph">

Временами будет трудно. Просидеть несколько часов над простым багом — суровая реальность. Эмоционально тоже может быть непросто — через это проходят все разработчики. Но в трудные моменты и происходит рост, который приносит ощущение победы.

</div>

<div class="paragraph">

Будьте готовы с головой погрузиться в теорию — впереди большие и насыщенные уроки. Чтобы освоить их, наберитесь сил и терпения.

</div>

## Что вы будете делать

  - Закрепите знания по вёрстке.
  - Освоите базу JS.
  - Узнаете, как управлять HTML и CSS с помощью JavaScript.
  - Научитесь исправлять ошибки в JavaScript. Это сэкономит много времени и нервов.

## Какие навыки вы получите

  - Впервые задействуете знания сразу о трёх технологиях: HTML, CSS и JavaScript.
  - Создадите адаптивную страницу в тренажёре.
  - С помощью массивов и циклов научитесь выводить контент на страницу автоматически.
  - Узнаете, как добавлять и удалять элементы со страницы в JavaScript.
  - В конце зальёте свой проект на GitHub. И его смогут увидеть другие пользователи.
