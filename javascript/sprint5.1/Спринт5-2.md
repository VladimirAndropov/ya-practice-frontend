# Вспомнить всё. Введение. От вёрстки к программированию

</div>

</div>

<div id="1eab02e9-909f-4198-9cca-fe65c1d3c1dd" class="section theory-viewer__block theory-viewer__block_type_markdown">

<div class="Markdown base-markdown base-markdown_with-gallery markdown markdown_size_normal markdown_type_theory full-markdown">

<div class="paragraph">

Весь первый модуль вы изучали разные аспекты вёрстки сайтов: располагали элементы с помощью флексбоксов и гридов, добивались корректной работы при любых разрешениях экрана, добавляли на сайт анимации, встраивали видео и делали многое другое, что позволяют языки HTML и CSS.

</div>

<div class="paragraph">

С помощью JavaScript вы добавите ещё больше интерактивности. Сайт сможет реагировать на действия пользователя: управлять тем, что именно он видит на странице; изменять, добавлять и удалять элементы.

</div>

<div class="paragraph">

Ваше знакомство с JavaScript состоялось в первом спринте, но за время интенсивного изучения HTML и CSS немудрено и забыть, о чём шла речь. Повторим пройденный материал в сжатой форме, а заодно закрепим его на новых примерах.

</div>

<div class="paragraph">

Мы говорили, что HTML — не язык программирования. Две последние буквы аббревиатуры HTML расшифровываются как Markup Language. А вот JavaScript как раз является языком программирования. Чтобы его понять, придётся перестроить мышление от разметки страницы к программному коду. Поначалу может быть трудно, но в конце концов вам понравится.

</div>

<div class="paragraph">
